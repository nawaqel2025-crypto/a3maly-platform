# A3MALY UI System Documentation

This folder contains the technical UI system reference used by the A3MALY migration.

## Documents

- `docs/system/components.md`: unified component patterns and usage.
- `docs/system/tokens.md`: color, typography, spacing, and layout tokens.
- `docs/system/layout-rules.md`: layout and responsiveness rules.
- `docs/system/migration-notes.md`: migration strategy, completed batches, and pending items.
