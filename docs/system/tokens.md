# Design Tokens

The following token groups are enforced globally via `app/globals.css`.

## Color Tokens

- Primary: `--a3-primary` (`#2563EB`)
- Primary Dark: `--a3-primary-dark` (`#1E40AF`)
- Secondary: `--a3-secondary` (`#0F172A`)
- Background: `--a3-background` (`#F8FAFC`)
- Surface: `--a3-surface` (`#FFFFFF`)
- Border: `--a3-border` (`#E2E8F0`)
- Text Primary: `--a3-text-primary` (`#0F172A`)
- Text Secondary: `--a3-text-secondary` (`#475569`)
- Status: `--a3-success`, `--a3-warning`, `--a3-danger`, `--a3-info`

## Typography Tokens

- Family: `Inter`
- Sizes: `--text-h1`, `--text-h2`, `--text-h3`, `--text-body`, `--text-small`, `--text-tiny`
- Weights: regular `400`, medium `500`, semibold `600`, bold `700`

## Spacing Tokens

- `--space-xs`, `--space-sm`, `--space-md`, `--space-lg`, `--space-xl`, `--space-xxl`
- Base mapping: 4, 8, 16, 24, 32, 48

## Layout Tokens

- Sidebar width: `--layout-sidebar-width` (260px)
- Header height: `--layout-header-height` (64px)
- Container max width: `--layout-container-max-width` (1440px)
- Card padding: `--layout-card-padding` (20px)
- Grid gap: `--layout-grid-gap` (16px)
- Section spacing: `--layout-section-spacing` (32px)
