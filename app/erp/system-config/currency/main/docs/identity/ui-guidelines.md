# A3MALY UI Guidelines

هذه الوثيقة توضح قواعد استخدام الهوية الجديدة في منصة أعمالي.

## المبادئ الأساسية
- البساطة أولًا
- الوضوح في كل عنصر
- استخدام المكونات الموحدة فقط
- عدم تكرار أي مكوّن
- الالتزام بالألوان الرسمية
- الالتزام بالمسافات الرسمية

## المكونات
يجب استخدام المكونات من:
`/components/ui/`

## الجداول
- رأس الجدول دائمًا بلون فاتح
- الصفوف تتفاعل عند المرور (hover)
- المسافات ثابتة

## الأزرار
- استخدام variant المناسب
- عدم إنشاء زر جديد خارج النظام

## النماذج
- الحقول موحدة
- المسافات ثابتة
- الرسائل واضحة
